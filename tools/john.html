1. What is John the Ripper

  • Tool used to crack passwords from hashes
  • Supports Linux passwords, ZIP/RAR, databases
  • Used in forensics and ethical hacking

⚠️ Important Rule

✔ Use only on your files or authorized systems

❌ Never crack real user passwords illegally

3. Installation

sudo apt update
sudo apt install john -y

🟢 Beginner Phase

Basic Syntax

john hash.txt

Create Hash File

nano hash.txt
5f4dcc3b5aa765d61d8327deb882cf99

Run John

john hash.txt

Show Password

john --show hash.txt

🟡 Intermediate Phase

Wordlist Attack

john --wordlist=/usr/share/wordlists/rockyou.txt hash.txt

Specify Format

john --format=raw-md5 hash.txt

Incremental Mode

john --incremental hash.txt

Rules (Smart Attack)

john --wordlist=rockyou.txt --rules hash.txt

🔴 Advanced Phase

Linux Password Crack

unshadow /etc/passwd /etc/shadow > combined.txt
john combined.txt

ZIP Crack

zip2john file.zip > ziphash.txt
john ziphash.txt

RAR Crack

rar2john file.rar > rarhash.txt
john rarhash.txt

Session Save & Resume

john --session=mycrack hash.txt
john --restore=mycrack

Parallel Cracking

john --fork=4 hash.txt

Pot File

john --show hash.txt

🧠 Real Workflow

1. Get hash
2. Identify type
3. Run wordlist
4. Apply rules
5. Try brute force

💡 Pro Tips

  • Start with rockyou.txt
  • Use rules for better results
  • Avoid brute force initially

🚀 Learning Plan

  • Beginner: Basic cracking + wordlists
  • Intermediate: Rules + formats
  • Advanced: System hashes + performance tuning

🔥 Final Truth

John doesn’t hack passwords.

👉 It guesses using patterns.

👉 Better wordlist = better results 💯
