💻 Kali Linux Guide

1. What is Kali Linux

Kali Linux is a Linux-based OS made for:

Cybersecurity 🔐
Ethical hacking 🧠
Digital forensics 🕵️

👉 Comes with 600+ tools like Nmap, Wireshark, Burp Suite

2. Why Kali Linux is Used

Penetration testing
Bug bounty hunting
Network analysis
Learning hacking legally

3. Requirements

4GB RAM (8GB better)
20GB Storage
VirtualBox installed

4. Download Kali Linux

👉 https://www.kali.org/get-kali/

Download Virtual Machine version (.7z) and extract it

5. Install Kali (VirtualBox)

Open VirtualBox → Add → Select .vbox file

Set RAM (4GB+) and CPU (2 cores)

Click Start

6. Login

username: kali
password: kali

7. Check Internet

ping google.com

8. Update Kali

sudo apt update && sudo apt upgrade -y

9. Basic Commands

ls
cd
pwd
mkdir
rm

10. Important Tools

Nmap → Port scanning

Burp Suite → Web testing

11. Practical Start
