📘 ZeroTrace User Guide

🧠 Introduction

ZeroTrace is a simulated cybersecurity training platform designed to help users learn real-world hacking tools and techniques in a safe and legal environment.

This platform is built for beginners, students, and aspiring ethical hackers who want hands-on experience.

⚠️ Legal & Ethical Disclaimer

This platform is strictly for educational purposes. Any misuse of cybersecurity knowledge for unauthorized access or malicious activities is illegal.

Violating cyber laws may result in serious legal consequences including fines and imprisonment.

🚀 Getting Started

Follow these steps:

  • Open the website
  • Login / Sign up
  • Enter dashboard
  • Explore sidebar

No prior experience required. Everything is beginner-friendly.

🧭 Dashboard Overview

  • Sidebar → Navigate labs & tools
  • Terminal → Execute commands
  • User Name → Personalized experience

💻 Terminal System

The terminal simulates a Kali Linux environment.

Basic Commands:

help clear ls whoami

Type commands and press Enter to execute.

⚙️ Tool Access via Commands

Open tools using commands:

open nmap open burp open wireshark

If command is invalid → system will show error.
