🧭 ZeroTrace Cybersecurity Roadmap (0 → ₹1L+)

🟢 Phase 1: Foundation (0–3 Months)

Goal: Learn basics + Linux + Networking

  • Networking: IP, DNS, TCP/UDP
  • Linux + Windows basics
  • Cybersecurity fundamentals

Tools: Kali Linux, Wireshark, Nmap

Commands:

ls
cd
pwd
mkdir
rm
ifconfig
ip a
nmap -sV <target>

Projects:

  • Network scanner using Nmap
  • Packet analysis using Wireshark
  • Kali Linux lab setup

Salary: ₹10K – ₹20K/month

🟡 Phase 2: Core Skills (3–6 Months)

  • HTML, JS, HTTP basics
  • OWASP Top 10
  • Basic vulnerabilities

Tools: Burp Suite, SQLMap, Gobuster

Projects:

  • Vulnerable website (DVWA clone)
  • SQL Injection demo
  • Password cracking (John the Ripper)

Salary: ₹20K – ₹40K/month

🔵 Phase 3: Specialization (6–12 Months)

Choose your path:

  • 🔐 Penetration Testing
  • 🧠 Digital Forensics (🔥 Recommended)
  • 🛡 SOC Analyst

Forensics Tools: Autopsy, FTK Imager, Volatility

Projects:

  • Investigate hacked system
  • Recover deleted files
  • Memory dump analysis

Salary: ₹40K – ₹1L/month

🔥 Must-Have Projects

  • ZeroTrace (your platform)
  • Bug bounty tracker
  • Custom vulnerable lab
  • Network monitoring tool
  • Forensics case study

💼 Internship Strategy

  • Create LinkedIn, GitHub, HackerOne
  • Post daily learning
  • Apply on Internshala & LinkedIn
  • Cold DM recruiters

📄 Resume Structure

  • Header (Name, Email, GitHub)
  • Objective
  • Skills (Nmap, Wireshark, Burp)
  • Projects (VERY IMPORTANT)
  • Certifications
  • Experience
  • Achievements

🧠 Daily Routine

  • 2 hrs → Learning
  • 2 hrs → Practice (Kali)
  • 1 hr → Build project
  • 30 min → LinkedIn post

💰 Salary Growth

Stage Time Salary
Beginner 0–6 months ₹10K–₹30K
Intermediate 6–12 months ₹30K–₹70K
Advanced 1–2 years ₹70K–₹1.5L
Pro 2+ years ₹2L+

⚡ Final Truth

If you only watch tutorials → ❌ No job If you build + practice daily → ✅ ₹1L job is real
