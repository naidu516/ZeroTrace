🛠️ Burp Suite Guide

💉 What is Burp Suite

Burp Suite is a powerful tool used for web application security testing.

It helps to:

  • Intercept requests 🔍
  • Modify traffic ⚙️
  • Find vulnerabilities 💥

⚙️ Installation

Usually pre-installed in Kali Linux.

sudo apt install burpsuite

🟢 Phase 1: Beginner

Proxy Setup:

  • Open Burp → Proxy → Turn ON Intercept
  • Set browser proxy to 127.0.0.1:8080

Intercept Request:

  • Capture request from browser
  • View headers and parameters

🟡 Phase 2: Intermediate

Repeater:

  • Send request to Repeater
  • Modify values and test manually

Intruder:

  • Automate attacks
  • Bruteforce parameters

Decoder:

  • Decode/encode data (Base64, URL, etc.)

🔴 Phase 3: Advanced

Scanner (Pro):

  • Automatically finds vulnerabilities

Extensions:

  • Add plugins from BApp Store

Session Handling:

  • Manage cookies & authentication

🧠 Real Workflow

  1. Capture request
  2. Send to Repeater
  3. Test manually
  4. Use Intruder
  5. Confirm vulnerability

💡 Pro Tips

  • Always test manually first
  • Use Burp with SQLMap
  • Understand requests deeply

🔥 Final Truth

👉 Burp doesn’t hack

👉 It helps you understand web apps

Understanding = Power 💯
