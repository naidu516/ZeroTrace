1. What is Nmap

Nmap is a network scanning tool used to:

  • Find open ports 🔓
  • Detect running services ⚙️
  • Identify OS 🧠
  • Used in ethical hacking, bug bounty, and network auditing

2. Install Nmap

sudo apt update
sudo apt install nmap -y

🟢 Beginner Phase

Basic Scan

nmap scanme.nmap.org

Scan Specific IP

nmap 192.168.1.1

Scan Multiple Targets

nmap 192.168.1.1 192.168.1.2
nmap 192.168.1.1-100

Scan Specific Port

nmap -p 80 192.168.1.1

Multiple Ports

nmap -p 22,80,443 192.168.1.1

All Ports

nmap -p- 192.168.1.1

🟡 Intermediate Phase

Service Version Detection

nmap -sV 192.168.1.1

OS Detection

nmap -O 192.168.1.1

Aggressive Scan

nmap -A 192.168.1.1

Fast Scan

nmap -F 192.168.1.1

Scan Without Ping

nmap -Pn 192.168.1.1

🔴 Advanced Phase

SYN Scan

sudo nmap -sS 192.168.1.1

UDP Scan

sudo nmap -sU 192.168.1.1

Script Scan

nmap --script vuln 192.168.1.1

Specific Script

nmap --script http-title 192.168.1.1

Scan Subnet

nmap 192.168.1.0/24

Save Output

nmap -oN result.txt 192.168.1.1

Firewall Evasion

nmap -f 192.168.1.1

🧠 Real Usage Flow

nmap -sn 192.168.1.0/24
nmap 192.168.1.5
nmap -sV 192.168.1.5
nmap -A 192.168.1.5

How Hackers Use Nmap

  • Find open ports
  • Identify services
  • Discover vulnerabilities
  • Plan attacks

⚠️ Rules

✔ Use only on authorized systems

❌ Don’t scan random websites

💡 Pro Tips

  • Start simple → then go advanced
  • Combine with Burp Suite & Wireshark
  • Always analyze results

🔥 Final Truth

Nmap doesn’t hack.

👉 It reveals information.

👉 Information = Power 💯
