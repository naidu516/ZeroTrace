1. What is John the Ripper

  • Password cracking tool 🔓
  • Used to test password strength
  • Supports multiple hash types
  • Widely used in cybersecurity & forensics

⚠️ Important Rule

✔ Use only on your own systems or labs

❌ Never crack real user passwords illegally

Install John

sudo apt update
sudo apt install john -y

🟢 Beginner Phase

Basic Crack

john hash.txt

Show Cracked Password

john --show hash.txt

Wordlist Attack

john --wordlist=rockyou.txt hash.txt

🟡 Intermediate Phase

Specify Format

john --format=raw-md5 hash.txt

List Formats

john --list=formats

Incremental Mode

john --incremental hash.txt

Single Crack Mode

john --single hash.txt

🔴 Advanced Phase

Custom Wordlist

john --wordlist=mylist.txt hash.txt

Rules (Mutation)

john --wordlist=rockyou.txt --rules hash.txt

Session Save

john --session=mysession hash.txt

Restore Session

john --restore=mysession

🧠 Real Usage Flow

1. Get hash
2. Identify type
3. Run wordlist attack
4. Try incremental
5. Analyze result

💡 Pro Tips

  • Use strong wordlists (rockyou.txt)
  • Always identify hash type first
  • Combine with hash-identifier tools
  • Be patient — cracking takes time

🔥 Final Truth

John doesn’t magically hack passwords.

👉 It guesses based on patterns.

👉 Strong passwords = harder to crack 💯
