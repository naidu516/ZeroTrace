🦈 Wireshark Full Guide

1. What is Wireshark

A tool to capture and analyze network traffic

Shows what data is going in and out of your system

Used in Cybersecurity, Networking, Forensics

🟢 PHASE 1: BEGINNER

⚙️ Open Wireshark

wireshark

🌐 Select Interface

wlan0 → Wi-Fi

eth0 → Ethernet

📡 Start/Stop Capture

Start → Double click

Stop → Red button

🔍 Screen Layout

  • Packet List
  • Packet Details
  • Packet Bytes

🎯 Filters

http
dns
tcp
ip.addr == 192.168.1.1

🧪 Practice

Capture DNS → visit google.com → filter dns

🟡 PHASE 2: INTERMEDIATE

🔁 TCP Stream

Right click → Follow TCP Stream

🧠 Protocols

  • DNS → domain to IP
  • TCP → connection
  • HTTP → website data

🔎 Analyze

Check IP, protocol, length

🔴 PHASE 3: ADVANCED

⚡ Detect Suspicious Activity

Unknown IPs, too many requests

🔥 Credentials (Lab Only)

Filter http → Follow TCP

🌐 HTTP vs HTTPS

HTTP readable

HTTPS encrypted

📊 Advanced Filters

ip.src == 192.168.1.5 && ip.dst == 192.168.1.1
dns.flags.response == 0
tcp.analysis.flags

🧰 Export Data

File → Export Objects

🧠 Use Cases

  • Debug network
  • Analyze malware
  • Investigate attacks

⚠️ Rules

✔ Use only on your network

❌ No illegal capture
